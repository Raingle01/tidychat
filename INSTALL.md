# TidyChat 安装教程（3分钟搞定）

## 📥 第一步：下载插件

点击下载：[tidychat-v1.0.0.zip](https://github.com/AeroXi/tidychat/releases/latest)

## 📂 第二步：解压文件

1. 找到下载的 `tidychat-v1.0.0.zip` 文件
2. **右键** → **解压到当前文件夹**（或双击解压）
3. 你会得到一个 `tidychat` 文件夹

> ⚠️ 记住这个文件夹的位置，不要删除！

## 🧩 第三步：安装到 Chrome

1. 打开 Chrome 浏览器
2. 在地址栏输入：`chrome://extensions/` 然后回车
3. 打开右上角的 **「开发者模式」** 开关
4. 点击左上角 **「加载已解压的扩展程序」**
5. 选择刚才解压的 `tidychat` 文件夹
6. 完成！🎉

![安装步骤示意图](https://user-images.githubusercontent.com/xxx/install-guide.png)

## ✅ 验证安装成功

- Chrome 右上角会出现 TidyChat 图标
- 打开 ChatGPT/Grok/Gemini/Claude/豆包 任意一个
- 页面右上角会出现绿色的折叠按钮

## 🎯 使用方法

| 操作 | 效果 |
|------|------|
| 点击绿色按钮 | 折叠/展开所有消息 |
| 点击折叠的消息 | 展开单条消息 |
| 拖动绿色按钮 | 移动到任意位置 |

## ❓ 常见问题

**Q: 安装后没看到按钮？**
- 确保你在支持的平台上（ChatGPT/Grok/Gemini/Claude/豆包）
- 刷新一下页面

**Q: 提示"无法加载扩展程序"？**
- 确保选择的是整个 `tidychat` 文件夹，不是里面的文件

**Q: 更新后怎么重新加载？**
- 回到 `chrome://extensions/` 页面
- 点击 TidyChat 卡片上的刷新按钮 🔄
